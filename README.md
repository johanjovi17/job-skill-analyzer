# Job Match Platform

A web app that helps users analyze how well they match a job based on their skills.

## Features

- User authentication
- Create and manage jobs
- Skill-based job matching (core feature)
- Match percentage + missing skills
- Responsive UI with loading states and toasts

## Tech Stack

- React
- Firebase (Auth + Firestore)
- React Router
- React Hot Toast

## How it works

Users select their skills → jobs have required skills → system compares both → shows match % and missing skills.
